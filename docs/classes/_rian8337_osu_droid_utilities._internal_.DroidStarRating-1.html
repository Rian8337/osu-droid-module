calculate

  • calculate(params: { map: <internal>.Beatmap; mods?: <internal>.Mod[]; stats?: <internal>.MapStats }): <internal>.DroidStarRating

  • Overrides StarRating.calculate

    • Defined in osu-rebalance-difficulty-calculator/typings/index.d.ts:343

    Calculates the star rating of the specified beatmap.

    The beatmap is analyzed in chunks of sectionLength duration. For each chunk the highest hitobject strains are added to a list which is then collapsed into a weighted sum, much like scores are weighted on a user's profile.

    For subsequent chunks, the initial max strain is calculated by decaying the previous hitobject's strain until the beginning of the new chunk.

    The first object doesn't generate a strain so we begin calculating from the second object.

    Also don't forget to manually add the peak strain for the last section which would otherwise be ignored.

    Parameters

    • params: { map: <internal>.Beatmap; mods?: <internal>.Mod[]; stats?: <internal>.MapStats }
      • map: <internal>.Beatmap

        The beatmap to calculate.

      • Optional mods?: <internal>.Mod[]

        Applied modifications.

      • Optional stats?: <internal>.MapStats

        Custom map statistics to apply custom tap multiplier as well as old statistics.

    Returns <internal>.DroidStarRating
